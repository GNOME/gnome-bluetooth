# GNOME Bluetooth


gnome-bluetooth is a helper library on top of the bluez daemon's D-Bus API.

Requirements
------------

- GTK
- bluez 5.51 or newer
- rfkill sub-system enabled in the kernel, and [accessible](https://github.com/systemd/systemd/pull/21605)
